#Website_challenge

##Gitの練習で作った初めての自作ウェブサイトです。
HTMLとCSSを使って作成しました。

##codejumpというサイト模写の練習サイトの教材を用いています。
URL
https://code-jump.com/

## 使い方
1. リポジトリをクローンします
2. GitHub上のページを「setting」→「pages」と切り替え、「visit site」をクリックします
3. Readmeのページが開いたら、見たいサイトのindex.htmlがあるフォルダのURLを、後ろにくっつけます。


## 使用技術
* HTML5
* CSS3
* Git / GitHub
